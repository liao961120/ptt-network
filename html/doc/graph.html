graph¶

pttnet.graph.Graph(count_edges_in, edge_condition=None, MG=None, edge_attrs_to_keep=['date', 'opinion'], node_path='data/network/nodes', edge_path='all_edges.jsonl')[source]¶

Generate nx.Graph from PTT comment data

Parameters
edge_conditiondict

Conditions for filtering edges from edge file. Valid only when argument MG is not None. Specified as a dictionary. Only values in a condition are kept. To keep all values of a condition, just leave out the key-value pair from the dictionary. The structure of edge_condition is as below:

{
  'date': {'2010-01-01', '2010-01-02', ...},
  'opinion': {'pos-pos', 'pos-neg', 'pos-neu', ...}
}

Data passed to loadMGraph().

count_edges_indict, optional

Criteria to reduce nx.MultiGraph into nx.Graph. The structure is similar to that of edge_condition. The same edges (having same nodes u, v) with values in a criterion are collapsed into 1 edge (with weight equals count). Edges in the MultiGraph not matching the criteria are pruned. Use emty dict {} to collapse all the same edges in the MultiGraph (regardless of criteria).

MGnx.MultiGraph, optional

nx.MultiGraph to start from. If specified, will use the MultiGraph (created from loadMGraph()) from memory instead of reading the node and edge files from disk (ignoring node_path and edge_path). By default None.

edge_attrs_to_keeplist, optional

Edge attributes to keep when reading file from disk, by default [‘date’, ‘opinion’]. Data passed to loadMGraph().

node_pathstr, optional

Path to the directory of node files, by default “data/network/nodes”. Data passed to loadMGraph().

edge_pathstr, optional

File path to edge file, by default “all_edges.jsonl”. Data passed to loadMGraph().

Returns
nx.Graph

Undirected graph with weighted edges.

Examples

First create a networkx.MultiGraph with loadMGraph() then reduce it to a networkx.Graph with Graph()

>>> conditions = {
...     'date': set(
...             (datetime.date(2010,1,1) + datetime.timedelta(days=x)).isoformat() 
...             for x in range(30)  # get 30 days comments starting from 2010-01-01
...         ), 
...     'opinion': set(['pos-pos', 'pos-neg', 'pos-neu', 'neg-pos', 'neg-neg', 'neg-neu'])
... }
>>> MG = loadMGraph(edge_condition=conditions, node_path="data/network/nodes", edge_path="data/all_edges.jsonl")
>>> 
>>> criteria = {
... 'opinion': ['pos-pos', 'neg-neg'],
... }
>>> G = Graph(count_edges_in=criteria, MG=MG)  # Further reduce graph

Alternatively, directly create a networkx.Graph from reading disk file:

>>> G = Graph(edge_condition=conditions, count_edges_in=criteria, node_path="data/network/nodes", edge_path="data/all_edges.jsonl")
